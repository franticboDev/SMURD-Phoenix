24.) Ghips

Info?
Ghips-ul este o construtie din ipsos si vata.

Cand il folosim?
Folosim ghipsul cand avem o fractură la o membră

Pentru ce folosim?
Il folosim pentru a fixa membra, o anumită perioada de timp, pentru a creste osul în poziția corecta

Info aplicare?
Se dezbraca pacientul, infasora membra in vata, se uda foiile, se aplica ipsosul.

Care sunt pasii? 
1) Se dezbraca pacientul 
2) Infasoara membra in vata pana unde este nevoie
3) Uda foile de ghips in apa calda
4) Cu foile de ghips infasori vata
5) Se pozitioneaza membra corect  ( se tine nemiscata timp de 3 min. )

Ce /me poti folosi?
/me Il dezbraca
/me infasoara vata pe ( Picior / Mana )
/me uda foile de ghips
/me aplica foile de ghips peste vata
